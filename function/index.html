    public async bootstrap() source 

    import {bootstrap} from '@ali/lib-js-sdk/src/bootstrap.js'

    启动 JS SDK

    1. 提供全局同步调用方法
    2. 初始化全局环境识别（基于 UserAgent），见 bootstrapGlobalEnv。
    3. 初始化 JS SDK 本体，见 bootstrapGlobalLib。
    4. 初始化全局事件通知机制，见 initGlobalEvents。
    5. 并发执行数组中已经存在的指令 FIFO
    6. 触发 ready 事件
